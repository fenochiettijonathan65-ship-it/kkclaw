# KKClaw Switch 重设计 Plan

## 一、设计理念
- **参考**: OpenAI/ChatGPT 界面风格
- **核心**: 黑白极简、流畅动画、小白友好
- **原则**: 信息清晰、操作直觉、零学习成本

## 二、文件模块化

### 当前状态
- `model-settings.html` ~972行（CSS + HTML + JS 全混在一起）

### 拆分方案
```
model-settings.html       ← 只保留 HTML 结构
model-settings.css        ← 所有样式（OpenAI 风格重写）
model-settings.js         ← 所有交互逻辑
```

## 三、OpenAI 设计规范

### 配色
| 用途         | 色值       | 说明              |
|:-------------|:-----------|:------------------|
| 背景         | `#FFFFFF`  | 纯白              |
| 次级背景     | `#F7F7F8`  | 浅灰（卡片悬停）  |
| 边框         | `#E5E5E5`  | 浅灰线            |
| 主文字       | `#000000`  | 纯黑              |
| 次级文字     | `#6B6B6B`  | 深灰              |
| 三级文字     | `#8E8EA0`  | 中灰              |
| 强调色       | `#10A37F`  | ChatGPT 绿        |
| 强调色悬停   | `#0E8C6F`  | 深绿              |
| 选中/激活    | `#ECECF1`  | 浅灰背景          |
| 危险         | `#EF4444`  | 红色              |
| 警告         | `#F59E0B`  | 橙色              |

### 字体
- 主字体: `Inter, -apple-system, sans-serif`
- 代码字体: `SF Mono, Consolas, monospace`
- 字号: 标题 18px, 正文 14px, 辅助 12px, 微小 11px

### 圆角
- 卡片: 12px
- 按钮: 8px
- 输入框: 8px
- 标签: 6px

### 间距
- 卡片间距: 8px
- 内边距: 16px
- 区块间距: 20px

### 4. Provider 卡片（核心改进）
```
┌──────────────────────────────────────┐
│ [🚗] kok8                    当前使用  │
│      gptclubapi.xyz · 21 models      │
│      ● Opus ━━━━━━━━░░░░░░ 剩余 94%  │
│                          [⏱] [✎]    │
└──────────────────────────────────────┘
```
- 白色背景 + 1px 灰色边框
- 当前使用的卡片：左边框 3px 绿色
- 悬停：浅灰背景 + 轻微阴影
- KKCLAW 配额条内嵌，不占额外空间
- 用 `data-idx` 代替 `id`，解决空格问题

### 5. 配额条
- 高度 4px，圆角 2px
- 背景 `#F0F0F0`
- 填充���:
  - 健康: `#10A37F` (ChatGPT 绿)
  - 警告: `#F59E0B` (橙色)
  - 危险: `#EF4444` (红色)
- 加载时脉冲动画

### 6. 按钮风格
- 主按钮: 绿色填充 `#10A37F`
- 次要按钮: 灰色边框 `#E5E5E5`
- 危险按钮: 红色文字 + 透明背景
- 悬停: 0.15s ease 背景变深
- 点击: 0.1s scale(0.97) 缩放

## 五、动画规范

| 动画           | 时长   | 缓动      | 场景            |
|:---------------|:-------|:----------|:----------------|
| 卡片悬停       | 0.2s   | ease      | 背景+阴影变化    |
| Tab 切换       | 0.2s   | ease      | 背景色过渡       |
| 配额条填充     | 0.6s   | ease-out  | 宽度从0到实际值  |
| 状态点脉冲     | 1.5s   | ease      | opacity 循环     |
| 按钮点击       | 0.1s   | ease      | scale(0.97)      |
| 面板切换       | 0.3s   | ease      | opacity + translateY |
| 搜索框聚焦     | 0.2s   | ease      | 边框色变化       |
| 卡片出现       | 0.3s   | ease-out  | 逐个 fadeInUp    |
| Toast 通知     | 0.3s   | ease      | 底部滑入/滑出   |
